# TASKS — EduCP Manager / CPManager

Gunakan file ini sebagai backlog detail per fase. Format checklist per fase:

## Fase XX — <Nama>
- [ ] Subtugas 1 (deskripsi spesifik)
- [ ] Subtugas 2
- [ ] Tes manual sesuai TESTING_CHECKLIST.md
- [ ] Update PROGRESS.md
- [ ] (Jika perlu) minta review agent terkait

Aturan:
- Pecah setiap fase menjadi subtugas kecil yang bisa di-commit terpisah.
- Tandai selesai hanya bila acceptance criteria fase terkait terpenuhi.
- Tautkan commit/PR pada subtugas bila relevan.
